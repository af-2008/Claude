## How to Use This Bank

### Instagram (max 30 tags per post)
- **Tier 1 (Boxing-specific):** 8–10 tags
- **Tier 2 (Fitness/transformation):** 4–5 tags
- **Tier 3 (Local/geographic):** 3–4 tags
- **Tier 4 (Viral/trend):** 3–4 tags
- Place hashtags BELOW the caption after a line break — never inline

### TikTok (3–5 tags only)
- Use 1–2 from Tier 1 + 1–2 from Tier 4
- No Tier 3 (local tags reduce discovery reach on TikTok)

### Facebook (3–5 tags only)
- Use from Tier 1 and Tier 3 only
- Facebook hashtags have limited discovery value — keep minimal

### YouTube Shorts
- No hashtags in caption — use in description only (3–5 max from Tier 1)
